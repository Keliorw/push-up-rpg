push-up-rpg — веб-демо детекции (не боевое приложение)

Использует TensorFlow.js MoveNet + настоящий RepDetector из проекта. Позиция подтверждается, когда камера видит прямой корпус (плечи–бёдра–колени) — гейт «планка». Повтор считается по вертикальному проседанию корпуса ИЛИ по сгибанию локтя. Если ног в кадре нет, включается запасной гейт по рукам (кисти ниже плеч + сгибание локтей). Строка внизу показывает живые сигналы — по ней удобно подбирать пороги в app/src/pose/config.ts.
